# Spam-email-detection using TF-IDF and Support Vector Machine

## Goal
Predicting whether an E-mail is spam or not.

## Dataset
-Spam email Dataset(Kaggle) or any emails.csv

## Requirements
- Python 3.8+
- pandas
- scikit-learn

## How to Run
1. Place 'emails.csv' in 'data/' .
2. Run 'spam-email-detection.py' .
